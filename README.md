# CESP Business

Landing page institucional da CESP Business, desenvolvida com React, Vite e Tailwind CSS.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Deploy

O projeto está configurado para publicação no GitHub Pages via GitHub Actions.

- Base pública: `/cesp-business/`
- Workflow: `.github/workflows/deploy.yml`
- Saída publicada: `dist/`

## Stack

- React 19
- Vite 8
- Tailwind CSS 4
- Framer Motion
- Lucide React
